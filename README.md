# Scraper Engine

Parallel scraper engine

## How to Run
Run app using run.sh file:

```
./run.sh --[clean | debug | pyvd | clean | help]
```

Close app using this command:
```
./run.sh --kill
```

### Prerequisites

```
* Firefox browser
```
## Built With
